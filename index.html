Developed balanced multi-way ( i.e with predefined fanout) tree called Sequence Condensation Tree (SCT) for fast Hierarchical partitioning of sequence dataset.The idea of condensation tree is to quickly partition the data into a group of similar object by organizing them in tree like structure which is inspired from well known BIRCH algorithm.
